# climateapi: Climate (and adjacent) data and functions for the C&C practice area

This packages combines a set of utilities for acquiring and processing
climate and climate-adjacent datasets under a consistent API. It takes
opinionated stances on how to manipulate raw data in an effort to
produce standard workflows that enable project teams to devote more time
to substantive analysis.
